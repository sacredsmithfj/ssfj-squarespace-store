Integration Instructions

To integrate these forms with your email marketing tool, follow the instructions below:

Mailchimp

1. Go to your Mailchimp account and create an audience.

2. Get the API key and replace the form action URL with your Mailchimp form URL.

Klaviyo

1. In Klaviyo, create a list to collect subscribers.

2. Use the API key to send data to Klaviyo.

ConvertKit

1. Create a form in ConvertKit and get the form action URL.

2. Replace the form action URL in the forms accordingly.
